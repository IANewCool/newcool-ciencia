# NewCool Ciencia - ANID Chile

Portal educativo sobre investigacion cientifica y fondos ANID en Chile.

## Demo

[https://ciencia.newcool.io](https://ciencia.newcool.io)

## Caracteristicas

- Directorio de centros de investigacion
- Informacion sobre fondos ANID y concursos
- Guia de Becas Chile
- Calculadora de tamano muestral
- Recursos para investigadores

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
